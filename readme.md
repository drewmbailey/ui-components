# UI Components

## Start

### Make sure a local server is installed:

```bash
npm install -g serve
```

### Install project files and build:

```bash
npm install
npm run build
```

### Start local server:

```bash
npm run serve
```

### Watch for changes:

```bash
npm run watch
```

### Start local server and watch for changes:

```bash
npm run start
```

View at http://localhost:3000.
